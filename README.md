#test-server
NodeJS + Express

##Available APIs

###Accounts
	GET api/accounts
returns accounts

###Roles
	GET api/roles
returns roles

##Server setup

###install dependencies:
	npm i
###run server:
	npm start
The server's API available on `http://localhost:3000`
###run tests:
	npm test
